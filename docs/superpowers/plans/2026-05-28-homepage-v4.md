> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rewrite the homepage hero, replace the three-service grid with an engagement ladder, update the founder section copy, upgrade JSON-LD schema, add an FAQ block, and open the robots.txt to AI crawlers.

**Architecture:** All changes are in the existing `app/(site)/page.tsx` + `page.module.css` route, two components (`FounderSection.tsx`, new `EngagementLadder.tsx`), `app/layout.tsx` for schema, and `app/robots.ts`. No new pages, no new routes. Sprint 1 only.

**Tech Stack:** Next.js 14 App Router, CSS Modules + global CSS variables, TypeScript. JSON-LD via `next/script` `<Script type="application/ld+json">`.
