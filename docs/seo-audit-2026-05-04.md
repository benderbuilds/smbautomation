## Issues NOT Fixed тАФ Manual Attention Required

### P2 Items

**1. `online-booking-automation-service-business` тАФ excerpt 3 chars over soft target**
- Current: 158 chars (soft target 155; hard limit 160)
- The excerpt contains three key stats that all pull weight. Further trimming possible ("Here's the system that captures that revenue." тЖТ "The system that captures that revenue." saves 4 chars), but not urgent.

**2. `speed-to-lead` тАФ H1 doesn't contain keyword phrase**
- H1: *"Why the First Business to Respond Wins 78% of the Time"*
- seoTitle: "Speed to Lead: Why Fastest Response Wins" тЬЕ (carries the keyword in title tag)
- **Recommendation:** Leave as-is. The H1 drives CTR; the seoTitle carries keyword. Acceptable GEO tradeoff.

**3. `dormant-customer-reactivation-automation` тАФ keyword in first 100 words**
- Primary keyword appears ~word 180. The churn-statistic hook is intentional and high-converting.
- **Recommendation:** Add one sentence after the opening stat: *"Automated win-back sequences are the direct fix тАФ and this guide shows you how to build one."* This can be done in a future sprint without disrupting the opening.

### P3 Items

**4. Blog index page title tag**
- Current: `title: 'Blog'` тЖТ renders "Blog | SMB Automation"
- **Recommendation:** Change to `'SMB Automation Blog: AI & Automation for Service Businesses'` (~55ch) for keyword presence on the index page. File: `app/(site)/blog/page.tsx`.

**5. `og:image` missing from blog post metadata**
- `generateMetadata` in `app/(site)/blog/[slug]/page.tsx` does not include `og:image`. AI assistants (Perplexity, ChatGPT Browse) and social platforms use OG images for rich previews.
- **Recommendation:** Add a dynamic OG image via `next/og` with post title + site branding. High GEO signal for zero-click visibility. Estimated effort: 3тАУ5 hours.

**6. `dateModified` missing from BlogPosting JSON-LD**
- Schema outputs `datePublished` but not `dateModified`. AI assistants weight freshness signals.
- **Recommendation:** Add `dateModified: post.date` to the BlogPosting JSON-LD in `app/(site)/blog/[slug]/page.tsx`. 30-minute change.

**7. `author` field hardcoded as Organization in BlogPosting**
- `author: { '@type': 'Organization', name: 'SMB Automation' }` тАФ a named Person author improves E-E-A-T signals, especially for healthcare and financial content.
- **Recommendation:** Add an `author` field to `PostMeta` (e.g., `author?: string`) and use `{ '@type': 'Person', name: post.author }` in the JSON-LD when set.
